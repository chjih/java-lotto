## 구현 기능 리스트 ##

- 입력
  - 로또 번호 입력
  - 보너스 번호 입력
  - 구입 금액 입력
    - 1000원 단위가 아닐 시 에러
- 출력
  - 발행한 로또 오름차순으로 정렬하여 출력
  - 당첨 내역 출력
  - 수익률 출력
- 기능
  - 결과 저장 클래스(WinningStat) 생성
  - Prize enum클래스 생성
    - 맞춘 개수에 따른 상금 저장
  - 로또 기능 수행할 LotteryMachine 클래스 생성
  - Randoms 사용하여 번호 추출
  - 일치 기능 구현
  - 단위 테스트 생성
